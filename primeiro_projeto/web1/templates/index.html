Aplicações Frontend

São responsável pela apresentação do software, chamado de client-side. É o código que roda no navegador (cliente). Trabalham com linguagens como HTML, CSS e JavaScript, além de frameworks e bibliotecas, como Angular, React, Vue.JS, etc. Não precisam conhecer como desenvolver código de backend, mas é importante que conheçam os fundamentos sobre arquitetura de software. Vale ressaltar que tanto o programador frontend quanto o backend precisam conhecer HTTP...
